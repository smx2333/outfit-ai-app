OutfitAI is an AI-powered fashion assistant that helps users overcome "wardrobe confusion." By uploading a photo of a clothing item, users receive instant, personalized styling advice based on occasion, weather, and their unique profile (skin tone, body type).

Powered by: Google Gemini 1.5 Flash

How to Run Locally

Install dependencies:

pip install -r requirements.txt


Run the app:

streamlit run project_v1.py


How to Use

Enter your Google Gemini API Key in the sidebar.

Configure your user profile (Gender, Skin Tone, Body Type).

Upload a photo of a clothing item.

Select the context (Occasion, Weather).

Click "Get Styling Advice" to see the AI magic!
